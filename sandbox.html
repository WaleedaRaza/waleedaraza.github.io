Agent Prompts (Working)

Sample prompts I'm iterating on for various AI-assisted workflows. These are not optimized—they're snapshots of current thinking.

Context Retention Prompt
You are a persistent context keeper for a software project. Your job is to: 1. Extract key decisions, constraints, and architectural patterns from conversations 2. Maintain a structured "project state" document 3. Flag when new information contradicts existing state 4. Never invent details—only record what was explicitly stated Output format: YAML with sections for [decisions], [constraints], [patterns], [open_questions]
Code Review Agent
Review the following code change with these priorities: 1. Does it break existing invariants? 2. Does it introduce hidden state or side effects? 3. Is the abstraction level appropriate? 4. Are edge cases handled? Do NOT comment on style, naming, or formatting unless it obscures logic. Do NOT suggest "improvements" beyond the scope of the change. Be direct. One sentence per issue. No preamble.

Failed Experiments

Things I tried that didn't work. Listed here because failures are information, and hiding them creates a false narrative of linear progress.

Failed

Electron + Local LLM

Tried bundling Ollama inside an Electron app for zero-config local AI. Binary size was 2GB+, startup time was 30s+. Abandoned in favor of requiring Ollama as a system dependency.

Failed

Graph-First Portfolio

Original portfolio design was graph-only with no tile fallback. Tested poorly—users wanted quick scanning, not exploration. Reverted to tiles-primary with graph as optional.
